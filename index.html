Variables

Variables allow you to specify widely used values in a single place, and then re-use them throughout the style sheet, making global changes as easy as changing one line of code. 
                    
@nice-blue: #5B83AD;
@light-blue: @nice-blue + #444;
.variables .example > div > div {
    color: @light-blue;
    border: thick solid @nice-blue;
}

.variables .example > div {
    border: thick solid @light-blue;
}

Mixins

Mixins allow you to embed all the properties of a class into another class by simply including the class name as one of its properties. It's just like variables, but for whole classes. Mixins can also behave like functions, and take arguments, as seen in the example bellow. 

                  
.rounded-corners (@radius: 5px) {
  border-radius: @radius;
  -webkit-border-radius: @radius;
  -moz-border-radius: @radius;
}

.thick-border (@color) {
    border: thick solid @color;
}

.mixins .example > div > div {
    .thick-border(@nice-blue);
    color: @light-blue;
    .rounded-corners;
}

.mixins .example > div {
    .thick-border(@light-blue);
    .rounded-corners(10px);
}

Nested Rules

Rather than constructing long selector names to specify inheritance, in Less you can simply nest selectors inside other selectors. This makes inheritance clear and style sheets shorter. 

                  
@orange: orange;
@dark-orange: @orange - #444;
.nested_rules {
    .example {
        div {
            border: thick solid @nice-blue;

            div:first-child {
                border: thick solid @orange;
            }
            div:last-child {
                border: thick solid @dark-orange;
            }
        }
    }
}

Operations

Are some elements in your style sheet proportional to other elements? Operations let you add, subtract, divide and multiply property values and colors, giving you the power to create complex relationships between properties. 

                  
@color: darkgreen;
@border-width: 5px;
.operations .example div {
    div:nth-child(1n+1) {
        background-color: @color / 3;
    }
    div:nth-child(1n+2) {
        background-color: @color / 2;
    }
    div:nth-child(1n+3) {
        background-color: @color;
    }
    div:nth-child(1n+4) {
        background-color: @color * 2;
    }
    div:nth-child(1n+5) {
        background-color: @color * 3;
    }
    div:nth-child(1n+6) {
        background-color: @color + #123;
    }
    div:nth-child(1n+7) {
        background-color: @color + #345;
    }
    div:nth-child(1n+8) {
        background-color: @color + #678;
    }
    div:nth-child(1n+9) {
        border: @border-width solid @color + #345;
    }
    div:nth-child(1n+10) {
        border: @border-width+3 solid @color + #123;
    }
    div:nth-child(1n+11) {
        border: @border-width+6 solid @color;
    }
}
